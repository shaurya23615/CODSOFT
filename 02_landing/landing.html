Build Websites Using HTML & CSS

HTML structures the content of a web page, while CSS styles it beautifully.

Basic HTML Page Structure

Every HTML page uses semantic tags to organize content:

<!DOCTYPE html>
<html>
  <head> ... </head>
  <body>
    <header> ... </header>
    <main> ... </main>
    <aside> ... </aside>
    <footer> ... </footer>
  </body>
</html>

Common HTML Elements

Header

The <header> contains the top section like logos or nav menus.

Section

The <section> divides content into meaningful sections.

Aside

The <aside> is for side content like tips or related links.

Footer

The <footer> contains page footer info, copyright, and links.

Tip

Use semantic HTML tags to improve accessibility, SEO, and readability of your code.
